Application description

Таблица сотрудников

Application который используется для отображения списка сотрудников в компании.

В нем можно:

  1. Сортировать спискок работников
  2. Добавлять сотрудников
  3. Писать дополнительную информацию про сотрудника / Обновлять ее
  4. Удалять(Увольнять)

В данном приложение я использовал БД и все функции используют отношения с ними.

Тут я использовал Raw Views для улучения понимания взаимодействий моделий и сути requests.

Для создания этого pet-app читал документацию и смотрел видео материалы, во многом помог Stack Overflow но по большей части использовал "тесты" непосредственно в самих функциях.
